# CardNumber-DeepOCR


I have developed a project using Keras and classic computer vision algorithms that takes as input various national identification cards and bank cards. The project goes through the following steps:

The input image is cropped similar to how CamScanner app works.
The project then detects whether the card is a national identification card or bank card.
The project finds the box containing the national identification card number or bank card number.
The numbers inside the box are predicted.
In essence, this project is similar to an OCR (Optical Character Recognition) system.
